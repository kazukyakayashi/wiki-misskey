    Les notes

    Attention !!

    Misskey étant en constante évolution, certains éléments peuvent avoir changé de place.

    Écrire une note

    Lorsque vous appuyez sur le bouton (en haut à droite), un nouveau formulaire apparaît.

    Astuce

    Pour plus de simplicité vous pouvez aussi activer directement le champ de saisie pour les notes au dessous de la Timeline .

    En haut

    1. : Fermer la fenêtre d'écriture de note
    2. Nombre de caractères autorisé par message1, avec un décompte visible en temps réel.
    3. : Visibilité de votre note (voir plus bas).
    4. Note : Envoyer la note.

    Au milieu :

    1. Écrivez votre note 2.

    En bas :

    1. : Ajouter un media (image ou audio), gère le glissé/déposé.
    2. : Faire un sondage.
    3. : CW (content warning), cache le texte avec un avertissement.
    4. : Ajouter un destinataire a la note.
    5. : Emojis

    Bien gérer la confidentialité de ses notes

    En haut de la fenêtre d'écriture de note, permet de gérer la visibilité de votre note, c'est a dire qui pourra la voir.

    1. : c'est a dire que cette note sera lisible de tout le fediverse.
    2. : Visible uniquement sur le fil local de l'instance, ne sera pas diffuser le le fil global de la fédération.
    3. : Visble uniquement par vos abonné(e)s.
    4. : Pour des messages privé avec juste les gens mentionné.
    5. : Confinement, permet de confiner une note en local et qui ne sera pas distribuée sur le fediverse.

    Information

    L'option est recommandé lorsque vous souhaitez avoir un compte privé (vous pourrez toujours publier des notes publique au besoin).

    Intéragir avec une note

    1. : Pour répondre à une note cliquez sur cela ouvrira la fenêtre de rédaction de note.
    2. : Pour re-note une note. Le re-note permet de republier une note publiée par vous même ou un autre utilisateur.
    3. : Pour ajouter une réaction.
    4. : Autres options.

    Information

    Pour annuler un re-note il vous suffit de cliquer sur la date du renote et d'annuler ce dernier.

    Les réactions

    Pour réagir a une note il vous suffit de cliquer sur le pour affiche la liste des émojis disponible.

    Bien que vous poussiez ajouter d'ici n'importe qu'elle émoji en réaction, vous pouvez configurer la liste des émojis disponible depuis les paramètres de compte pour plus de rapidité.

    Autres options

    D'autres options sont disponibles via le menu , comme l'ajout de la note en favoris ou de copier le lmien de cette dernière.

    1. La limite de caractères est différente pour chaque cas. Il peut y avoir une différence dans le décompte d'un caractère entre l'affichage du caractère et la reconnaissance du programme. ↩

    2. Si vous fermez le formulaire pendant que vous écrivez, vous pouvez reprendre votre note là où vous l'aviez laissé. ↩
